# E-Commerce OOP Java Console Application (ArrayList Version)

## Features
- Admin: Add/update products, view all products, view all orders.
- User: Register/login, add products to cart, view cart, place orders.
- Cart: ArrayList-based cart with quantity update for existing products.

## How to Run
1. Open terminal in src folder.
2. Compile all files:
   javac *.java
3. Run the main class:
   java ECommerceApp
